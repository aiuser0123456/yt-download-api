YouTube Downloader API

Welcome to the YouTube Downloader API! This API allows you to extract information and download links from YouTube videos.

API Endpoint

/download?url=[YOUTUBE_URL]

Example Usage

To get information about a YouTube video, use the following URL format:

{{ url_for('download_video', _external=True) }}?url=https://www.youtube.com/watch?v=dQw4w9WgXcQ

Response Format

The API returns a JSON response with the following structure:

{
  "title": "Video Title",
  "channel": "Channel Name",
  "duration": 212,
  "thumbnail": "https://img.youtube.com/vi/VIDEO_ID/maxresdefault.jpg",
  "description": "Video description...",
  "upload_date": "20230101",
  "view_count": 1234567,
  "like_count": 12345,
  "formats": [
    {
      "format_id": "22",
      "format_note": "hd720",
      "resolution": "1280x720",
      "extension": "mp4",
      "filesize_approx": 45678901,
      "url": "https://direct-url-to-video.mp4",
      "vcodec": "avc1.64001F",
      "acodec": "mp4a.40.2",
      "fps": 30
    },
    ...
  ]
}

How to Use

  1. Find the YouTube video you want to download
  2. Copy the video URL
  3. Replace [YOUTUBE_URL] in the endpoint with your video URL
  4. Access the URL in your browser or application
  5. Use the direct URL from the format you prefer to download the video
